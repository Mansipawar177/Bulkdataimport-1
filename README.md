# Bulk Data Import & Validation System

## Description
This is a Spring Boot application for bulk data import and validation.

## Technologies Used
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Swagger OpenAPI

## Features
- CSV/Excel File Upload
- Data Validation
- Bulk Import
- Exception Handling
- REST APIs
- DTO
- Repository Pattern
